# 杜小梅个人作品集网站

## 项目简介

这是一个基于 Vue 3 + Vite 构建的个人作品集网站，用于展示个人项目、技能和工作经验。网站采用现代化的设计风格，包含响应式布局、动画效果和交互体验，展现个人专业能力和项目成果。

## 技术栈

- **前端框架**：Vue 3
- **构建工具**：Vite
- **CSS 动画**：Animate.css
- **开发语言**：JavaScript
- **项目结构**：单页应用 (SPA)

## 安装和运行

### 安装依赖

```bash
npm install
```

### 开发环境运行

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

### 预览生产版本

```bash
npm run preview
```

## 项目亮点

1. **现代化设计**：采用深色主题、紫色渐变和玻璃态效果，营造专业现代的视觉体验
2. **性能优化**：使用 Vite 构建工具，实现快速开发和构建
3. **动画效果**：集成 Animate.css 实现流畅的动画效果，提升用户体验
4. **响应式布局**：适配各种设备屏幕，确保在不同设备上的良好展示
5. **瀑布流布局**：项目列表采用瀑布流布局，美观且高效利用空间
6. **交互体验**：丰富的交互效果，包括悬停动画、模态框和滚动触发动画

## 主要项目展示

1. **网络安全实训靶场前端系统**：企业级应用，支持网络环境构建、攻防演练、技能评估等功能
2. **指挥控制系统前端**：3D 可视化项目，集成任务规划、态势监控、载荷管理等功能
3. **报名系统**：Uni-app 开发的多端应用，支持小程序和 H5 端
4. **VS Code 插件开发**：Git Commit Lint 插件，提升代码提交质量
5. **企业官网**：响应式企业官网，展示公司信息和产品服务
